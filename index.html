🧪 ผังวิเคราะห์แคทไอออน (Qualitative Analysis)

รูปแบบเต็มจอ A4 แนวนอน (มีไอออน 1 ชนิด: $\mathrm{Ag^+}$, $\mathrm{Pb^{2+}}$, $\mathrm{Ni^{2+}}$, $\mathrm{Fe^{3+}}$, $\mathrm{Mn^{2+}}$)

  • 📦 สารตัวอย่าง (ของแข็ง)
    การหาตัวทำละลาย:
    1. น้ำ $1\ \mathrm{mL}$ (อุ่น $2-3$ นาทีถ้าไม่ละลาย)
    2. เปลี่ยนสารใหม่ ใช้ $2.0\ \mathrm{M}\ \mathrm{HCl}$ (อุ่นถ้าไม่ละลาย)
    3. เปลี่ยนสารใหม่ ใช้ $2.0\ \mathrm{M}\ \mathrm{HNO_3}$ (อุ่นถ้าไม่ละลาย)
    * ห้ามใช้สารหลอดเดิมมาเติมกรดเด็ดขาด ต้องตักผงของแข็งใหม่ทุกครั้งที่เปลี่ยนตัวทำละลาย
    ตอนที่ 1: หยด $6.0\ \mathrm{M}\ \mathrm{HCl}$ $1-2$ หยด
    (หยดเพิ่มจนตกตะกอนสมบูรณ์ แล้วปั่นเซนตริฟิวจ์)
    💡 ถ้าหยดแล้วขุ่นขาวทันที = เป็นแก๊ง Ag/Pb
    💡 ถ้าหยดแล้วยังใสปิ๊ง = เป็นแก๊ง Ni/Fe/Mn
    $$\begin{aligned} \mathrm{Ag^+} + \mathrm{Cl^-} &\rightarrow \mathrm{AgCl}_{(s)} \\ \mathrm{Pb^{2+}} + 2\mathrm{Cl^-} &\rightarrow \mathrm{PbCl}_{2(s)} \end{aligned}$$
    • ☁️ ตะกอนสีขาว
      ($\mathrm{AgCl}_{(s)}$, $\mathrm{PbCl}_{2(s)}$)
      ตอน 2: เติมน้ำ $1\ \mathrm{mL}$
      ➔ อุ่นร้อน $2-3$ นาที
      ➔ รีบปั่นขณะร้อนจัด
      คนด้วยแท่งแก้วขณะอุ่น
      ⚠️ ต้องรีบแยกตอนน้ำยังร้อน ไม่งั้นตะกอนตะกั่วจะตกผลึกกลับมาใหม่
      • 💧 สารละลายใส
        ($\mathrm{Pb^{2+}}$ ละลายในน้ำร้อน)
        ทดสอบ: หยด $0.1\ \mathrm{M}\ \mathrm{K_2CrO_4}$ $2-3$ หยด
        * สามารถยืนยันผลเพิ่มโดยการนำตะกอนเหลืองไปละลายใน $6.0\ \mathrm{M}\ \mathrm{NaOH}$ (ตะกอนเหลืองจะหายไปเป็นสารละลายใส)
        • 🟡 ตะกอนสีเหลือง
          ➔ สรุปพบ $\mathrm{Pb^{2+}}$
          $$\mathrm{Pb^{2+}} + \mathrm{CrO_4^{2-}} \rightarrow \mathrm{PbCrO}_{4(s)}$$
      • ☁️ ตะกอนขาวเดิม
        ($\mathrm{AgCl}_{(s)}$ ไม่ละลายน้ำร้อน)
        ล้างตะกอนด้วยน้ำร้อน $2\ \mathrm{mL}$ ทิ้งก่อน
        ทดสอบ: หยด $6.0\ \mathrm{M}\ \mathrm{NH_3}$ $1\ \mathrm{mL}$ (ตะกอนจะละลายใส)
        ➔ หยด $6.0\ \mathrm{M}\ \mathrm{HNO_3}$ ทีละหยด (เทสจนเป็นกรด)
        * กรดจะไปทำลายพันธะแอมโมเนีย ทำให้ตะกอนขาวกลับมาอีกครั้ง
        • ☁️ ตะกอนขาวกลับมา
          ➔ สรุปพบ $\mathrm{Ag^+}$
          $$\begin{aligned} &[\mathrm{Ag(NH_3)_2}]^+ + \mathrm{Cl^-} + 2\mathrm{H^+} \\ &\rightarrow \mathrm{AgCl}_{(s)} + 2\mathrm{NH_4^+} \end{aligned}$$
    • 💧 สารละลายใส
      (ไม่มีตะกอน $\mathrm{Ag/Pb}$)
      ตกตะกอน: หยด $6.0\ \mathrm{M}\ \mathrm{NaOH}$ จนเป็นเบส (+เผื่อ $0.5\ \mathrm{mL}$)
      ➔ เติม $3\%\ \mathrm{H_2O_2}$ $1\ \mathrm{mL}$ ➔ อุ่นร้อน $2$ นาที ➔ ปั่น ➔ ทิ้งน้ำใส เก็บตะกอน
      การอุ่น: เทใส่ชามกระเบื้องนำไปอุ่นบน hot plate จนปริมาตรลดลงเหลือครึ่งเดียว เพื่อบังคับให้ไอออนตกตะกอนให้หมด
      • 🪨 ตะกอนสีต่างๆ
        (เขียว $\mathrm{Ni}$ / น้ำตาล $\mathrm{Fe}$ / ดำ $\mathrm{Mn}$)
        ตอน 3: ล้างตะกอนด้วยน้ำแล้วทิ้งน้ำก่อน
        ➔ ละลายด้วย $6.0\ \mathrm{M}\ \mathrm{HNO_3}$ $10$ หยด + $6.0\ \mathrm{M}\ \mathrm{HCl}$ $5-6$ หยด ➔ อุ่นจนใส
        ➔ เติม $4.0\ \mathrm{M}\ \mathrm{NH_4Cl}$ $4$ หยด ➔ ปรับเบสด้วย $15\ \mathrm{M}\ \mathrm{NH_3}$ ➔ เติม $3\%\ \mathrm{H_2O_2}$ $1\ \mathrm{mL}$ ➔ ปั่นแยก
        * ขั้นตอนนี้จะดึง $\mathrm{Ni}$ ให้ออกไปอยู่ในสารละลาย ส่วน $\mathrm{Fe, Mn}$ จะยังคงเป็นตะกอน
        • 💧 สารละลายสีฟ้า
          ($[\mathrm{Ni(NH_3)_6}]^{2+}$)
          ทดสอบ: หยด $\mathrm{DMG}$ $2-3$ หยด
          เขย่าหลอดให้แรง เพื่อให้แอมโมเนียเชิงซ้อนทำปฏิกิริยากับ Dimethyl glyoxime ได้อย่างสมบูรณ์
          • 🔴 ตะกอนสีแดง
            ➔ สรุปพบ $\mathrm{Ni^{2+}}$
            $$\begin{aligned} &[\mathrm{Ni(NH_3)_6}]^{2+} + 2\mathrm{H_2DMG} \\ &\rightarrow \mathrm{Ni(HDMG)}_{2(s)} + 4\mathrm{NH_3} \end{aligned}$$
        • 🪨 ตะกอนน้ำตาล/ดำ
          ($\mathrm{Fe(OH)_3}$ หรือ $\mathrm{MnO_2}$)
          ล้างตะกอนด้วยน้ำ $1\ \mathrm{mL}$ ทิ้งก่อน
          ➔ ละลายด้วย $6.0\ \mathrm{M}\ \mathrm{HCl}$ $1\ \mathrm{mL}$ + $6.0\ \mathrm{M}\ \mathrm{HNO_3}$ $2$ หยด
          ➔ อุ่นจนใส ➔ แบ่งสารละลายใส่ $3$ หลอด
          ต้องต้มอุ่นให้ตะกอนละลายใสกลายเป็นน้ำทั้งหมด ก่อนที่จะแบ่งใส่หลอดทดลองย่อย
          • 🧪 เทส $\mathrm{Fe^{3+}}$ (หลอด 1,2)
            หลอด 1: หยด $1.0\ \mathrm{M}\ \mathrm{KSCN}$ $4-5$ หยด
            หลอด 2: หยด $0.1\ \mathrm{M}\ \mathrm{K_4Fe(CN)_6}$ $2-3$ หยด
            * ในห้องสอบ สามารถเลือกทำหลอดใดหลอดหนึ่งก็ได้ เพราะให้ผลสรุปเหมือนกัน
            • 🩸 สีแดงเลือดนก
              (หรือตะกอนน้ำเงินปรัสเซียน)
              ➔ สรุปพบ $\mathrm{Fe^{3+}}$
              $$\mathrm{Fe^{3+}} + \mathrm{SCN^-} \rightarrow [\mathrm{FeSCN}]^{2+}$$
          • 🧪 เทส $\mathrm{Mn^{2+}}$ (หลอด 3)
            หยด $6.0\ \mathrm{M}\ \mathrm{HNO_3}$ $2$ หยด แล้วอุ่น
            ➔ เติมผง $\mathrm{NaBiO_3}$ $1$ ช้อนเล็ก ➔ เขย่า
            👀 ทริค: ต้องจับตาดูสีทันทีหลังเติมผง $\mathrm{NaBiO_3}$ ลงไป เพราะสีด่างทับทิมจะสลายตัวและหายไปเร็วมาก
            • 🟣 สีม่วงแดงด่างทับทิม
              (เปอร์แมงกาเนต - สีจางเร็วมาก)
              ➔ สรุปพบ $\mathrm{Mn^{2+}}$
              $$\mathrm{Mn^{2+}} \xrightarrow{\mathrm{NaBiO_3}} \mathrm{MnO_4^-}$$
